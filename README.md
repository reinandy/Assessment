# Assessment    

### Installation

1. Open in cmd or terminal app and navigate to this folder
2. Run following commands

```bash
composer install
```

```bash
cp .env.example .env
```

```bash
php artisan key:generate
```

```bash
php artisan serve
```

And navigate to generated server link (http://127.0.0.1:8000)
